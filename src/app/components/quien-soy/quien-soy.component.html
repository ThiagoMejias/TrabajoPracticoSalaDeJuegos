Mi juego: Adivina el numero

El juego consiste en adivinar el numero, para ellos tendras dos opciones de juego. Modo facil: Deberas adivinar el numero mientras se te dice si el numero es mas grande o mas pequeño. Modo Dificil: Deberas adivinar el numero mientras se te dice si estas mas caliente o mas frio. Mientras Puntuacion: Mientras mas rapido adivines mas puntos sumaras, ademas el modo dificil te dara el doble de puntos que el facil
